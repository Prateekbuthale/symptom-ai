## âš™ï¸ Key Design Decisions

**1. 100% AI Reasoning (No rule-based logic)**

- Decision: All symptom interpretation and recommendations are LLM-driven
- Why: Ensures flexible, up-to-date, and â€œhuman-likeâ€ interactive experiences
